# Telegram-подобный UI и функционал

## Что реализовано

### 🎨 UI в стиле Telegram

1. **Список чатов (ChatListScreen)**
   - Дизайн как в Telegram
   - Аватары пользователей с цветными инициалами
   - Показ последнего сообщения
   - Время последнего сообщения
   - Счетчик непрочитанных сообщений
   - Поиск по чатам
   - Индикатор онлайн статуса

2. **Окно чата (ChatScreen)**
   - Фон как в Telegram (#E5DDD5)
   - Пузыри сообщений с правильными скруглениями
   - Сообщения пользователя справа (зеленые)
   - Сообщения других слева (белые)
   - Аватары для групповых чатов
   - Имена авторов в групповых чатах
   - Время отправки
   - Статусы сообщений (отправлено, доставлено)

3. **Компоненты**
   - `Avatar` - аватар с инициалами и индикатором онлайн
   - `ChatListItem` - элемент списка чатов
   - `MessageBubble` - пузырь сообщения

4. **Экран профиля**
   - Аватар пользователя
   - Имя и email
   - Меню настроек
   - Кнопка выхода

### 🔧 Backend функционал

1. **Личные чаты**
   - Поддержка типа чата (private/group)
   - Автоматическое создание личных чатов
   - Endpoint `/chats/private` для создания/получения личного чата

2. **Последние сообщения**
   - Обновление `lastMessage` в чате при отправке
   - Обновление `lastMessageAt`
   - Сортировка чатов по последнему сообщению

3. **Непрочитанные сообщения**
   - Счетчик непрочитанных для каждого пользователя
   - Автоматическое увеличение при новом сообщении

4. **Ответы на сообщения**
   - Поддержка `replyTo` в сообщениях
   - Отображение цитируемого сообщения

5. **Обновления в реальном времени**
   - Событие `chat_updated` для обновления списка чатов
   - Автоматическое обновление при новом сообщении

## API изменения

### Новые endpoints

- `POST /chats/private` - Создать/получить личный чат
  ```json
  {
    "userId": "other_user_id"
  }
  ```

### Обновленные endpoints

- `GET /chats` - Теперь возвращает:
  - `type` - тип чата (private/group)
  - `lastMessage` - последнее сообщение
  - `lastMessageAt` - время последнего сообщения
  - `otherUser` - для личных чатов
  - `unreadCount` - количество непрочитанных

### Socket события

- `chat_updated` - обновление чата (новое сообщение, изменения)
- `send_message` - теперь поддерживает `replyTo`

## Использование

### Создать личный чат

```javascript
const res = await api.post("/chats/private", {
  userId: "other_user_id"
});
```

### Ответить на сообщение

```javascript
socket.emit("send_message", {
  chatId: "chat_id",
  text: "Ответ",
  replyTo: "message_id"
});
```

## Стилизация

Цвета как в Telegram:
- Фон чата: `#E5DDD5`
- Мои сообщения: `#DCF8C6`
- Сообщения других: `#FFFFFF`
- Акцент: `#3390EC`
- Текст: `#000000`
- Вторичный текст: `#707579`
